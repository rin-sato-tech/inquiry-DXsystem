# テストチェックリスト

## 1. 目的

本ドキュメントは、社内問い合わせ管理システムの動作確認項目を整理するものです。

Ver.1では、CSV取込、DB保存、問い合わせ一覧表示、新規登録、ステータス更新、期限超過判定、集計、Tableau用CSV出力を確認対象としました。
Ver.2では、以下の追加機能も確認対象に含めます。

- 要対応アラート
- FAQ候補管理
- カテゴリ別入力フォーム
- 依頼者向け確認画面
- Ver.2追加集計
- Tableau出力列の拡張
- DB移行処理
- Ver.2スモークテスト

## 14. テスト結果まとめ

主要機能であるCSV取込、DB保存、問い合わせ一覧表示、新規登録、ステータス更新、期限超過判定、集計、Tableau用CSV出力が動作することを確認しました。
また、Ver.2で追加した要対応アラート、FAQ候補管理、カテゴリ別入力フォーム、依頼者向け確認、Ver.2追加集計、Tableau出力列の拡張についても、基本動作を確認しました。Ver.2追加機能によって、Ver.1の基本機能が壊れていないことも確認しました。

本格運用に向けては、ログイン認証、権限管理、通知機能、FAQ公開ページ、FAQ検索、添付ファイル、操作履歴などを今後の課題とします。
