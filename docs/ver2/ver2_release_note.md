# Ver.2 変更点まとめ

## 目的

Ver.2では、Ver.1の問い合わせ管理機能を拡張し、問い合わせ削減・自己解決支援につながる機能を追加した。

## 追加機能

| 機能                   | 内容                                                        |
| ---------------------- | ----------------------------------------------------------- |
| 要対応アラート         | 期限超過・本日期限・期限間近・担当者未設定などを一覧化      |
| FAQ候補管理            | 完了済み問い合わせからFAQ候補を登録・更新・解除             |
| カテゴリ別入力フォーム | カテゴリに応じた追加情報を入力                              |
| 依頼者向け確認画面     | 問い合わせID・依頼者名で問い合わせ状況を確認                |
| Ver.2集計              | アラート、FAQ候補、追加情報入力率、依頼者向け表示件数を集計 |
| Tableau出力更新        | Ver.2追加列をTableau用CSVに出力                             |

## Ver.2で追加した主なファイル

| ファイル                 | 内容                      |
| ------------------------ | ------------------------- |
| `src/alerts.py`          | 要対応アラート判定        |
| `src/faq.py`             | FAQ候補管理               |
| `src/category_fields.py` | カテゴリ別入力項目定義    |
| `src/requester_view.py`  | 依頼者向け表示・検索      |
| `src/migrate_db.py`      | Ver.2追加カラムのDB移行   |
| `src/smoke_test_ver2.py` | Ver.2機能のスモークテスト |

## 本格運用に向けた課題

- ログイン認証
- 権限管理
- FAQ公開ページ
- 通知機能
- 添付ファイル
- 操作履歴
- PostgreSQL移行
