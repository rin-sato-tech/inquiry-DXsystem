# Ver.2 ポートフォリオ説明用メモ

## 30秒説明

Ver.1では、社内問い合わせを一元管理し、一覧表示、登録、更新、期限超過判定、集計、Tableau出力までを実装しました。

Ver.2では、要対応アラート、FAQ候補管理、カテゴリ別入力フォーム、依頼者向け確認画面を追加しました。

これにより、問い合わせ管理だけでなく、対応漏れ防止、確認往復削減、FAQ化による問い合わせ削減、依頼者側の進捗確認まで含めた簡易DXアプリに発展させています。

## 特に見てほしい点

- 業務課題から機能を設計したこと
- DB移行を考慮してVer.2追加カラムを実装したこと
- アラートやFAQ候補など、実運用を想定した改善を加えたこと
- Streamlitだけでなく、Tableau用CSV出力まで整えたこと
- テストチェックリスト、保守手順書、学習ログまで整理したこと
