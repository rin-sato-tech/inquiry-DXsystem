# 操作手順書

## 1. 目的

この文書は、社内問い合わせ管理システムの基本操作を説明するものです。
対象画面は、Streamlitで作成した以下の4画面です。

- 問い合わせ一覧
- 新規登録
- ステータス更新
- 集計・CSV出力

## 2. アプリの起動方法

プロジェクトフォルダで以下を実行します。

```bash
cd ~/projects/inquiry-dx-system
source .venv/bin/activate
streamlit run app.py
```

ブラウザでアプリが開きます。

## 3. 問い合わせ一覧画面

### 3.1 目的

登録済みの問い合わせを一覧で確認する画面です。

### 3.2 表示される内容

主に以下の項目が表示されます。

- 問い合わせID
- 受付日
- 依頼者
- 部署
- 受付経路
- カテゴリ
- 優先度
- 希望期限
- 担当者
- ステータス
- 期限超過
- 問い合わせ内容
- 対応内容

### 3.3 絞り込み

以下の条件で絞り込みできます。

- 部署
- カテゴリ
- 担当者
- ステータス
- 優先度
- 期限超過のみ

絞り込みを行うと、一覧だけでなくKPIも絞り込み後の値に変わります。

## 4. 新規登録画面

### 4.1 目的

新しい問い合わせを登録する画面です。

### 4.2 入力項目

| 項目               | 必須 | 内容                       |
| ------------------ | ---- | -------------------------- |
| 依頼者名           | 必須 | 問い合わせを行った社員名   |
| 部署               | 必須 | 依頼者の所属部署           |
| 受付経路           | 必須 | フォーム、メール、電話など |
| カテゴリ           | 必須 | 問い合わせ分類             |
| 小分類             | 任意 | より細かい分類             |
| 優先度             | 必須 | 高・中・低                 |
| 希望期限           | 必須 | 対応希望日                 |
| 担当者             | 任意 | 管理部担当者               |
| ステータス         | 必須 | 初期状態は未対応           |
| 問い合わせ内容     | 必須 | 問い合わせの詳細           |
| 不足情報・確認事項 | 任意 | 追加で確認が必要な情報     |

### 4.3 登録時の動作

登録すると、問い合わせIDが自動発行されます。形式は以下です。

```text
REQ-YYYYMMDD-001
```

登録後は、問い合わせ一覧画面で確認できます。

## 5. ステータス更新画面

### 5.1 目的

既存問い合わせの担当者、ステータス、対応内容、作業時間を更新する画面です。

### 5.2 更新できる項目

- 担当者
- ステータス
- 完了日
- 対応内容
- 記録・管理上の問題
- 管理作業時間
- 実対応時間

### 5.3 完了日の扱い

完了日は、ステータスが「完了」のときだけ設定できます。ステータスが以下の場合、完了日は保存されません。

- 未対応
- 対応中
- 情報待ち
- 承認待ち

完了済みの問い合わせを完了以外のステータスに戻した場合、完了日はクリアされます。

## 6. 集計・CSV出力画面

### 6.1 目的

問い合わせ状況を集計し、Tableau連携用CSVを出力する画面です。

### 6.2 表示されるKPI

- 問い合わせ件数
- 未完了件数
- 期限超過件数
- 平均対応日数
- 管理作業時間
- 実対応時間
- 問い合わせ関連時間

### 6.3 表示される集計

- 期限超過案件一覧
- カテゴリ別件数
- 担当者別件数
- 部署別件数
- ステータス別件数
- 受付経路別件数
- 優先度別件数
- 担当者別作業時間
- カテゴリ別作業時間
- カテゴリ別平均対応日数

### 6.4 Tableau用CSV出力

以下の2通りでCSV出力できます。

- CSVをダウンロード
- `data/tableau_output.csv` に保存

出力CSVには、以下のような派生列が含まれます。

- 期限超過フラグ
- 完了フラグ
- 未完了フラグ
- 対応日数
- 管理作業時間
- 実対応時間
- 合計作業時間

## 7. 終了方法

ターミナルで以下を押します。

```text
Ctrl + C
```
